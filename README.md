# Auto-Man Driving School

Website for Auto-Man Driving School in Geelong and surrounding suburbs.  
Provides lesson information and online bookings via Google Calendar.

## Features
- Responsive design using custom CSS (theme.css)
- Driving Test Package with warm-up and test-day car use
- Google Calendar integration for bookings
- Client portal with booking management
- Contact information and links to social media
